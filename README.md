# Restaurant-Page #
This is one of the Intermediate JavaScript Projects provided by "The Odin Project".
This project utilizes Webpack functionality and the use of the Module system. This
project was a lot of fun to work on because I got to see first hand the power of
having multiple JavaScript files to not only seperate my code but give each file
it's own purpose.
